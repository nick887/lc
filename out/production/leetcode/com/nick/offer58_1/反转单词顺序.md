## 反转单词顺序
### 解法1：双指针
```java
    public String reverseWords(String s) {
        s=s.trim();
        String[] a= s.split("\\s");
        StringBuffer stringBuffer=new StringBuffer();
        for (int i = a.length; i >-1 ; i--) {
        stringBuffer.append(a[i]+" ");
        }
        return stringBuffer.toString().trim();
        }
```
